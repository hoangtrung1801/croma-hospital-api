## Run

+ Create file .env with
```env
MONGODB_URL = <URL_TO_MONGODB_SERVER>

PORT = <PORT>

TOKEN_SECRET = <SECRET_STRING_FOR_AUTHENTICATION>
```

+ Install packages
```sh
yarn install
```
+ Install redis and run it
```sh
redis-server
```
+ Run
```sh
yarn start
```
